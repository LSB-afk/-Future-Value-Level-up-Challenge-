## 국토교통 분야 실증서비스 모델 구현 기획서

### 서비스 모델명

MoveValue: 주거-이동 통합 생활권 매칭 서비스

### 분야

국토교통 데이터 융합, 생활권 추천, 부동산·교통 연계 서비스

### 서비스 제공 대상

공공 및 민간 모두 해당. 공공은 지자체·정책 부서·공공 데이터 활용 사업을 대상으로 하고, 민간은 부동산 플랫폼·MaaS 플랫폼·기업 이주 지원 서비스·일반 시민을 대상으로 한다.

### 실증서비스 아이디어 모델 개요

MoveValue는 사용자의 월 주거 예산, 출퇴근 목적지, 가구 유형, 생활 인프라 우선순위를 입력받아 후보 생활권을 추천하는 서비스다. 단순 월세나 매매가가 아니라 주거비, 이동시간, 환승 편의, 생활 SOC 접근성, 안전·환경 지표를 함께 계산한다.

핵심 기능은 생활권 랭킹, 항목별 점수 설명, 후보지 비교, 정책·플랫폼용 API 제공이다. 시민은 실제 생활비와 이동 부담을 고려해 거주지를 고를 수 있고, 지자체와 민간 플랫폼은 생활권 수요와 취약 지점을 데이터 기반으로 파악할 수 있다.

### 배경 및 필요성

주거 선택 과정에서 시민은 가격, 직장 접근성, 대중교통, 생활시설을 각각 따로 검색해야 한다. 이 때문에 월세는 낮지만 통근시간과 교통비가 높아지는 선택, 환승 부담이 큰 선택, 생활 인프라 접근성이 떨어지는 선택이 발생한다.

국토교통 분야에는 실거래가, 지오코딩, 도시철도 역사, 교통 데이터 오픈마켓 등 활용 가능한 데이터가 이미 존재한다. 그러나 이 데이터가 시민의 실제 의사결정 화면까지 연결되는 서비스는 제한적이다. MoveValue는 분산된 국토교통 데이터를 생활권 단위로 연결해 국민이 체감하는 새로운 가치를 만드는 것을 목표로 한다.

주 시장은 청년 1인 가구, 신혼부부, 출퇴근 직장인, 교통약자 가족이며, 민간 확장 시장은 부동산 플랫폼 API, 기업 이주 지원, 지자체 생활권 분석 리포트다.

### 차별성

기존 부동산 서비스는 매물 가격과 위치 검색 중심이다. MoveValue는 거주 후 매월 발생하는 이동 부담과 생활 접근성을 비용화·점수화한다. 또한 추천 결과를 블랙박스로 제시하지 않고, 주거비·통근·교통 접근성·생활 SOC·환경 항목별 근거를 함께 보여준다.

공공데이터만으로 최소 기능을 구현할 수 있고, 민간 이동 데이터나 플랫폼 데이터가 붙으면 정밀도를 높일 수 있는 단계형 구조라는 점도 차별점이다. 시민용 서비스, 민간 API, 공공 정책 리포트가 하나의 밸류체인 안에서 연결된다.

### 현황 및 구체화 방안

현재 작업공간에는 서울시 2025 전월세 실데이터를 집계한 API 기반 웹 프로토타입이 구현되어 있다. 사용자는 예산, 출퇴근 목적지, 가구 유형, 우선순위를 조정하고 후보 생활권의 종합점수와 세부 점수, 주거비 집계 근거를 확인할 수 있다.

선정 후에는 국토교통부 실거래가 API, VWorld 지오코더, 대중교통 경로 API, 전국도시철도 역사정보, 국가교통 데이터 오픈마켓 데이터를 추가 어댑터로 연결한다. 최종 산출물은 웹 프로토타입 1식, 데이터 연계 모듈 1식, 생활권 점수 산정 API 1식, 지자체·플랫폼용 리포트 예시 1식으로 구성한다.

서비스 간 연계는 부동산 플랫폼에는 생활권 점수 API, 지자체에는 생활권 취약지 리포트, 데이터 공급자에는 데이터 품질 피드백과 수요 리포트를 제공하는 방식으로 설계한다.

### 서비스 구현을 위한 협력 방안

협력 후보는 세 그룹이다. 첫째, 공공데이터 제공기관과 국토교통 데이터 오픈마켓을 통해 실거래가, 교통, 좌표 데이터를 확보한다. 둘째, 부동산 플랫폼 또는 지역 중개 네트워크와 협력해 사용자 접점을 확보한다. 셋째, 지자체·교통 데이터 사업자와 협력해 생활권 진단과 이동 데이터 보정을 수행한다.

역할 분담은 MoveValue가 데이터 결합, 점수 산정, 화면/API 개발을 맡고, 데이터 공급자는 원천 데이터 및 품질 정보를 제공하며, 수요처는 서비스 실증과 사용자 피드백을 제공하는 구조다.

### 기대효과 및 해결하고자 하는 사회 현안

시민은 주거비만 보고 결정할 때 놓치기 쉬운 통근시간, 교통비, 생활 인프라 접근성을 한 화면에서 비교할 수 있다. 이는 청년·신혼부부·직장인의 주거 선택 시행착오를 줄이고, 교통약자 가족에게 더 적합한 생활권 탐색을 돕는다.

지자체는 생활권별 교통·주거 부담을 시민 관점으로 파악해 정책 우선순위를 정할 수 있다. 민간 플랫폼은 국토교통 데이터 기반 부가 서비스를 붙여 차별화할 수 있으며, 데이터 공급자는 실제 서비스 수요와 연결되어 데이터 산업 생태계 활성화에 기여한다.
