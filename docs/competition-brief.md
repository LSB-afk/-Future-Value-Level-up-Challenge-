# 2026 국토교통 서비스 발굴 경연 확인 요약

확인일: 2026-06-11

## 공모전 핵심 조건

- 공모명: 2026 국토교통 서비스 발굴 경연, Future Value Level-up Challenge
- 목적: 국토교통 분야 서비스 밸류체인 기반 서비스 모델 발굴을 통한 데이터 산업 생태계 활성화 및 국민이 체감하는 새로운 가치 창출
- 개최기간: 2026년 5월 ~ 9월
- 접수기간: 2026년 5월 18일(월) ~ 6월 19일(금)
- 참가대상: 국토교통 데이터 기반 국민체감형 서비스 모델을 구현하여 국토교통 데이터 산업 생태계 활성화에 기여하고자 하는 자
- 제출처: `kalda@kalda.or.kr`
- 문의: `070-8211-1278`
- 시상: 대상 1점, 우수상 등 총 7점
- 특전: 국토교통부 협력형 분석과제 모델 선정 시 후보등록

## 제출물 관점에서 중요한 해석

- 단순 아이디어보다 선정 후 기간 내 서비스 제작·개발 가능한 최종 산출물 형태를 제시해야 한다.
- 국민체감 가치, 국토교통 데이터 사용 근거, 서비스 간 연계 방법, 협력 밸류체인 구상이 필요하다.
- 제출 자료의 저작권은 참가자에게 있으나, 수상작은 협의 후 공개·홍보 활용될 수 있다.
- 타 공모전 수상작, 저작권 침해, 도용, 모방, 허위 기재는 취소 사유다.

## 로컬 양식에서 확인한 작성 항목

- 참가 신청서: 기관명, 사업 분야, 담당자, 연락처, 이메일, 서비스 모델명, 서비스 사업 분야, 산출물, 예산, 기간, 참여기업
- 구현 기획서: 모델명, 분야, 제공 대상, 개요, 배경 및 필요성, 차별성, 현황 및 구체화 방안, 협력 방안, 기대효과, 기간, 예산
- 서약서: 규정 준수, 저작권·도용 금지, 공개·홍보 활용 협의, 소스 사용권 확인, 허위기재 책임
- 개인정보 동의서: 신청자 개인정보 수집·이용 동의 필수
