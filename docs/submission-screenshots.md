| --- | --- | --- | --- | --- | --- |
| `01-main-screen.png` | 메인 화면 | 지도 기반 주거 의사결정 화면 제시 | 서비스 접속 | 조건 입력 패널과 서울 지도/단지 라벨 표시 | MoveValue는 첫 화면에서 조건 입력과 지도 탐색을 함께 제공해 주거 후보지를 공간적으로 비교하게 한다. |
| `02-condition-input.png` | 사용자 조건 입력 화면 | 예산, 목적지, 가구 유형, 가중치 입력 | 회사 주소와 선호 조건 입력 | 추천 계산에 필요한 파라미터 구성 | 사용자는 월 주거 예산, 출퇴근 목적지, 가구 유형, 통근/주거비/SOC/안전 가중치를 조정할 수 있다. |
| `03-recommendation-results.png` | 생활권 추천 결과 화면 | 조건에 맞는 후보 랭킹 제공 | `매칭하기` 클릭 | 단지별 추천 순위, 월세, 통근시간 표시 | 추천 결과는 가격만이 아니라 통근시간과 생활권 점수를 결합한 순위로 제공된다. |
| `04-map-based-recommendation.png` | 지도 기반 추천 화면 | 후보 단지를 지도에서 탐색 | 지도 마커 확인 | 추천 단지와 주변 단지가 라벨로 표시 | 지도 위 단지 라벨은 사용자가 후보 위치와 주변 주거 공급을 동시에 확인하도록 돕는다. |
| `05-property-dashboard.png` | 아파트/건물 상세 정보 대시보드 | 단지 상세 검토 | 추천 단지 선택 후 `아파트 정보` 탭 클릭 | 기본정보, 가격, 거래추이, 위험 신호 표시 | 단지 선택 시 기본정보, 가격, 거래 추이, 전세 위험 신호를 하나의 상세 대시보드에서 확인한다. |
| `06-price-risk-metrics.png` | 실거래가·공시가격·전세가율 화면 | 가격과 위험 지표 확인 | 상세 패널 스크롤 | 매매가, 전세가, 월세, 공시가격, 전세가율, 추이 그래프 표시 | 가격 정보는 실거래 API 연계 가능 상태와 추정/폴백 여부를 함께 표시해 데이터 신뢰 범위를 드러낸다. |
| `07-candidate-comparison.png` | 후보 지역/단지 비교 화면 | 여러 후보 비교 | 즐겨찾기 단지 추가 후 비교 패널 열기 | 가격, 통근시간, 전세가율, 위험도, SOC 비교표 표시 | 사용자는 관심 단지를 저장하고 가격, 통근, 위험 신호, 생활 SOC를 표로 비교할 수 있다. |
| `08-jeonse-risk-check.png` | 전세 위험 신호 점검 결과 화면 | 계약 전 주의 요소 안내 | 상세 패널에서 위험 신호 확인 | 전세가율, 공시가격 대비 보증금, 변동성, 등기부 확인 항목 표시 | 전세 위험 신호는 법적 판정이 아니라 계약 전 확인해야 할 주의 요소와 근거를 안내한다. |
| `09-ai-agent-qa.png` | AI Agent 질의응답 화면 | 데이터 근거 기반 설명 | 전세 안전성 질문 입력 | 가격/통근/위험/입지 근거 그룹으로 답변 | AI Agent는 단순 챗봇이 아니라 가격, 이동, 생활권, 위험 신호 데이터를 근거로 설명형 답변을 제공한다. |
| `10-mobile-responsive.png` | 모바일 반응형 화면 | 모바일 사용성 검증 | 모바일 폭에서 접속 | 입력 패널과 추천 결과가 1열로 재배치 | 모바일에서도 예산 입력, 추천 결과, 지도 탐색 흐름이 유지되도록 반응형 UI를 적용했다. |
